# Is Everyone Safe - Crisis Communication Website

This is the marketing website for Is Everyone Safe, a crisis and incident communication platform that serves employees, students, event attendees, and citizens.

## Features

- Marketing homepage with hero, features, and use cases
- Dedicated solution pages for different audiences (Workplace, Education, Events, Government)
- Blog and news system with UK-specific content
- SEO optimisation with structured data and sitemaps
- Responsive design with accessibility features

## Development

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
\`\`\`

## Deployment

This site is designed to be deployed on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy

The site will automatically rebuild when you push changes to GitHub.

## SEO Features

- Dynamic sitemap generation including blog posts
- Structured data (JSON-LD) for better search engine understanding
- Open Graph tags for social media sharing
- Optimised meta descriptions and titles
- Semantic HTML and accessibility features

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui
- **Deployment:** Vercel
- **TypeScript:** Full type safety

## Content Management

Blog posts are managed in `lib/blog-posts.ts`. To add or edit blog posts, update this file with the new content.
